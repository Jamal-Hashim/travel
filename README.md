# 🌍 Travel Website

A modern and responsive **Travel Website** built using **HTML, CSS, and JavaScript**.  
This project is designed to showcase travel destinations, tour packages, and booking options in an attractive and user-friendly interface.

The website provides a smooth browsing experience with clean design, animations, and a fully responsive layout for all devices.

---

## 🌟 Features

- 🏠 **Home Page (Index)** with featured destinations  
- 📖 **About Page** with company or travel agency details  
- 🧳 **Packages Page** showcasing travel plans and offers  
- 📝 **Booking Page** for trip reservations  
- 🎨 Beautiful and modern UI design  
- ⚡ Smooth animations and hover effects  
- 📱 Fully responsive design (Mobile + Desktop)  
- 🧭 Easy navigation menu  

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactivity and dynamic features  
- **Font Awesome** – Icons  
- **Responsive Web Design**
